NIF Toolbar¶

The NIF Toolbar is a suite of graphical user interfaces (GUIs) and functions, intended to simplify the management of experimental control parameters for running behavioral neuroscience experiments involving non-human primates (NHPs). NIF Toolbar is written in Matlab and designed to interface with commonly used Matlab extensions for behavioral neuroscience.

As the name suggests, the NIF Toolbar has been developed specifically for use in the NIF (Neurophysiology Imaging Facility) and SCNI (Section on Cognitive Neurophysiology and Imaging) at the National Institutes of Health. It is provided to NIF users as a more open-ended alternative to the excellent NIMH Monkey Logic extension for Matlab, which is also available for use in the NIF and is worth considering for your research needs.

Basic Overview¶

The NIF Toolbar is flexibly designed to be of benefit to both novice and experienced researchers. For the novice with little programming experience, the NIF Toolbar’s graphic environment offers an intuitive way of getting simple visual experiments up and running quickly. For this purpose, a series of commonly used experimental templates are provided, including:

  • Eye-tracker calibration

  • Presentation of static images (e.g. fMRI ‘face patch localizer’)

  • Presentation of movie clips (e.g. fMRI block design)

  • Presentation of abstract graphics (e.g. dot motion, gratings, retinotopic mapping stimuli)

For the advanced user, NIF Toolbar provides a fast and user-friendly means of editing and organizing the many variables involved (via GUIs), which can then be optionally accessed in your own experimental code through the Params super-structure. NTB sub-functions allow common tasks that may occur throughout an experiment (e.g. reward delivery) to be executed with a single line of code, thus minimizing the amount of code required to write new experiments.

Requirements¶

All systems running the NIF Toolbar comprise of the following:

Hardware:

  • PC with 32GB RAM, 1TB SSD

  • NVidia Geforce GTX 1080 (8GB)

  • VPixx DataPixx 2 interface

  • Video-based eye tracking PC with analog output

  • ref

    DataPixx Interface box (optional)

  • Neurophysiology recording system (optional)

Software:

  • Ubuntu 16.04

  • NVidia drivers 450.66.

  • Matlab R2020a+

  • PsychToolbox Version 3.0.16

  • DataPixx Toolbox (optional)

  • PLDAPS Version 4.3.0 (optional)

Installation¶

The NIF Toolbar can be downloaded from the GitHub repository. The computer system should already be configured as outlined in System Installation.
