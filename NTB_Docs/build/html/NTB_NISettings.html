NTB National Instruments Settings¶

The National Instruments (NI) Settings GUI allows the user to quickly set channel mappings for NI data acquisition (DAQ) devices in order to route analog and digital signals. National Instruments DAQs can be used as an alternative to the DataPixx DAQ. All variables controlled by the NI settings GUI are saved to the NI field of the Params structure.
