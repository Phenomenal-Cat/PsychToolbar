NTB Actions Tab¶

The Actions tab of the NIF Toolbar GUI allows the user to execute various actions based on the current settings. The most important action.

Icon

Action

Brief Description

Run selected experiment

Runs the experiment function that is currently selected in the NTB Session panel’s ‘Experiment’ dropdown menu.

Stop current experiment

Stops the currently running experiment (without closing the onscreen window).

Run eye calibration

Runs an eye calibration routine based on the settings selected in the eye calibration settings GUI.

Deliver manual reward

Manually delivers liquid reward to the subject based on the settings selected in the reward settings GUI.

Play audio

Play the audio file(s) selected in the audio settings GUI.
