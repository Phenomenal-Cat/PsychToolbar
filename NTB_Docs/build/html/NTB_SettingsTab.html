NTB Settings Tab¶

The Settings tab of the NIF Toolbar GUI allows the user to launch various settings apps to quickly load, edit and save variables related to specific aspects of behavioral experiments and associated hardware. The variables controlled by each app in the Settings tab are saved to a unique field of the Params structure, as listed in the table below. Click on the icon or name of each setting GUI to view it’s documentation, or click on the output structure to see the list of fields within each settings output structure.

Icon

Setting GUI

Output Structure

Brief Description

Display

Params.Display

Configure visual display geometry and appearance for both subject and experimenter.

Datapixx

Params.DPx

Set analog and digital IO channels for the DataPixx interface box.

Eye tracking

Params.Eye

Set parameters for eye-tracker calibration in order to convert analog eye signals into screen coordinates.

Reward

Params.Reward

Set parameters for liquid reward delivery schedule.

Keyboard

Params.Keys

Set experimenter keyboard shortcuts that are active while the experiment is running.

Image pres.

Params.Images

Set parameters for an image presentation experiment, including the paths of the images to be presented.

Movie pres.

Params.Movies

Set parameters for a movie presentation experiment, including the paths of the movies to be presented.

TDT ephys.

Params.TDT

Set parameters for communication with Tucker Davis Technologies (TDT) neurophysiology recording systems.

Open Ephys

Params.OE

Set parameters for communication with Open EPhys (OE) neurophysiology recording systems.
